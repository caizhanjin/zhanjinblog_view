# zhanjinblog_view

zhanjinBlog个人博客前端代码

+ [vue.js](https://cn.vuejs.org/)
+ [vue-element](https://element.eleme.cn/#/zh-CN)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
